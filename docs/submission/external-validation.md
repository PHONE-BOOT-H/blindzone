# 외부 타당성 검증 — 정부 "응급의료취약지" 공식 지정과의 일치

> BlindZone이 사고·사망·직선거리 기반으로 독립적으로 지목한 robust 사각지대(인제·옹진)가, 전혀 다른 방법론을 쓰는 **정부의 공식 응급의료취약지 지정**과 같은 방향을 가리키는지 대조한다. 같은 곳을 가리키면, BlindZone 지수가 임의 가중치의 산물이 아니라 현실의 구조적 취약을 포착한다는 **수렴 타당도(convergent validity)** 근거가 된다.

## 1. 왜 외부 검증이 필요한가

BlindZone 위험지수는 자체 정의한 가중합 지수다. 내부적으로 가중치 민감도(126 시나리오)·사망률 보정으로 강건성을 검증했지만(weight-sensitivity.md), 이는 모두 **같은 데이터 안에서의 내적 검증**이다. "이 지수가 현실의 응급 취약을 실제로 맞히는가"는 외부의 독립 기준과 대조해야 답할 수 있다.

가장 강력한 외부 기준은 **정부가 전혀 다른 방법으로 지정한 응급의료취약지 목록**이다. 방법이 다른데 결론이 같으면, 두 방법이 공통으로 포착하는 실체(구조적 응급 취약)가 있다는 뜻이다.

## 2. 정부 응급의료취약지 지정 제도

- **근거**: 보건복지부 고시 「응급의료분야 의료취약지 지정」 (최신본 고시 제2024-261호, 2024.12.30 시행). 응급의료에 관한 법률에 따라 주기적으로 지정·고시.
- **지정 기준**: 지역 내 인구 중 **지역응급의료센터로 30분 이내 도달 불가능**하거나 **권역응급의료센터로 60분 이내 도달 불가능**한 인구가 **30% 이상**인 시군구.
- **규모**: 전국 약 98~99개 시군구 (시점에 따라 변동). 주로 군 지역·인구 15만 미만 도농복합시.
- **방법론 대비**: 정부 기준은 **실제 도로망 기반 도달시간 + 인구 분포**를 쓴다. BlindZone은 **TAAS 사고다발 + 사망 + 응급기관 직선거리**를 쓴다. **입력 데이터도, 계산 방식도 독립적이다.**

## 3. BlindZone robust 사각지대 vs 정부 지정

### 옹진군 — 정부 취약지로 직접 확인됨

- BlindZone: 사고 0건인데도 응급기관 직선거리 75.3km(전국 최대)로 가중치·사망률 보정 양쪽에 강건한 1순위 robust 사각지대.
- 정부 지정: 인천 옹진군은 서해 도서지역으로, 응급의료취약지 관련 다수 공식 출처(2023.12 비대면진료 확대 대상 98개 취약지 등)에서 직접 언급된다. 군 내 응급실 보유 종합병원이 없고 의료가 보건지소·진료소 중심.
- **결론: BlindZone과 정부 지정이 같은 곳을 가리킨다.**

### 인제군 — 구조적 근거 강함 (별표 직접확인 권장)

- BlindZone: 사고 1건, 응급기관 직선거리 24.2km, 사망률 보정엔 민감하나 거리 요인은 잔존.
- 정부/공식 근거: 강원특별자치도청 응급의료기관 현황상 인제군은 **관내 응급의료기관(권역/지역응급의료센터·지역응급의료기관)이 하나도 없는 군**(고성·양양과 함께). 산악지형이라 권역센터(춘천) 도달 시간이 길어 30분/60분·30% 기준을 충족하는 구조. 강원 군지역 응급의료 접근성이 46.4% 수준이라는 도 자료도 있음.
- **결론: 관내 응급의료기관 0개 = 구조적 취약. 고시 별표 원문에서 인제군 행 직접 확인 시 정량 근거로 확정.**

## 4. 해석 — 수렴 타당도

- BlindZone(사고+거리 기반 단순 지수)과 정부(도달시간+인구 기반 정교한 기준)는 **방법이 완전히 다른데 인제·옹진을 공통으로 지목**한다.
- 이는 BlindZone 상위 사각지대가 임의 가중치의 우연한 산물이 아니라, 독립적 정부 기준도 포착하는 **실재하는 구조적 응급 취약**임을 시사한다.
- 동시에 BlindZone의 추가 가치도 분명하다: 정부 취약지 목록은 "지정/비지정" 이분 라벨이지만, BlindZone은 **252개 시군구 연속 위험지수 + 기여요인(SHAP) + What-if 정책 시뮬레이션**을 제공한다. 즉 "어디가 취약한가"(정부)에 더해 "왜 취약하고, 무엇을 바꾸면 얼마나 나아지나"(BlindZone)를 답한다.

## 5. 한계 (정직 고지)

- **별표 전체 목록(98~99개 시군구) 텍스트를 자동 추출하지 못했다.** 고시 PDF 별표가 표/이미지 형태라, 인제·옹진의 별표 내 명시적 행은 **한태영이 PDF를 직접 열어 최종 확인**해야 한다(아래 액션). 본 문서는 옹진=직접 언급 확인, 인제=구조 근거 강함 수준으로 정직하게 표기한다.
- 정량 일치율(정부 취약지 98개 ∩ BlindZone top-N 겹침 비율)은 별표 목록 텍스트 확보 후 산출 가능. 확보 시 weight-sensitivity 결과와 교차하면 강력한 한 줄이 된다.
- 119 구급통계(2023, raw 보유)는 외부검증 후보였으나, 경기·대구 소방본부 누락(15개 시도본부만)·이송시간 컬럼 부재·소방서-시군구 매핑 불완전으로 본 검증에서는 제외했다.
