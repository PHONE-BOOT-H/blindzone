# 수요(고령) 변수 교차분석 + 수혜인구 — robust 사각지대의 구조적 취약 입증

> BlindZone 위험지수는 공급측(사고·사망·응급 접근성)만 본다. "사고 0~1건인데 왜 위험한가"라는 직관적 반론에 답하려면, 수요측(인구·고령) 취약을 결합해야 한다. 행정안전부 주민등록 고령인구현황을 시군구 단위로 매칭해, robust 사각지대(인제·옹진)가 수요측에서도 취약한지 검증한다.

## 1. 데이터

- **행정안전부 주민등록인구 — 고령 인구현황 (2026년 4월)**, 전국 시군구. (jumin.mois.go.kr, 비로그인 CSV)
- 시군구 252개 전부 매칭(`backend/scripts/merge_population.py`, 시도 prefix + 시군구명 정규화 — `data_pipeline`과 동형). 65세이상비율 = 65세이상 인구 / 총인구.
- **위험지수에 직접 통합하지 않고 교차 해석 변수로 사용** — 가중치 항목을 늘리면 또 다른 임의성이 생기므로, robust 결론(공급측)과 수요측을 분리해 교차한다.

## 2. 결과 1 — robust 사각지대는 수요측에서도 취약하다

전국 고령비율 중앙값 24.8% / 평균 27.5% (min 10.8%, max 49.9%).

| 시군구 | 사고건수 | top10_share | 고령비율 | 유형 |
|---|---:|---:|---:|---|
| 인제군 | 1 | 0.91 | **28.0%** | 사각지대 |
| 송파구 | 2,012 | 0.86 | 18.8% | 대도시 |
| 동대문구 | 1,756 | 0.82 | 21.0% | 대도시 |
| 대구 북구 | 1,365 | 0.75 | 20.8% | 대도시 |
| 달서구 | 1,409 | 0.74 | 21.4% | 대도시 |
| **옹진군** | **0** | 0.73 | **37.2%** | 사각지대 |
| 영등포구 | 1,468 | 0.72 | 19.5% | 대도시 |
| 중랑구 | 1,207 | 0.64 | 23.9% | 대도시 |

- **대도시 robust 6곳 평균 고령 20.9% ↔ 사각지대(인제·옹진) 평균 32.6%.**
- 옹진군 37.2%(전국 53위, 상위 21%), 인제군 28.0%(전국 104위, 평균 이상). 대도시 robust는 모두 전국 중앙값(24.8%)보다 낮다.
- **핵심 해석**: 대도시 robust는 사고가 많지만 고령은 낮다 — 이미 알려진 사고다발지다. 인제·옹진은 사고가 0~1건인데 고령이 1.5배 높다. 즉 **공급측(응급거리 멀다) + 수요측(고령 많다) 양쪽에서 취약**하다. "사고가 적은 게 아니라, 응급 수요가 큰데 접근이 구조적으로 막힌 곳"이라는 BlindZone의 핵심 메시지가 수요 데이터로 뒷받침된다.

## 3. 결과 2 — 수혜 인구 (정책 환산)

What-if 시뮬레이션(가상 응급거점 1곳 추가 → 응급거리 단축)에 인구를 결합하면, 거리 단축의 잠재 수혜 대상을 환산할 수 있다.

| 사각지대 | 응급거리 변화 | 잠재 수혜 총인구 | 고령 인구 |
|---|---|---:|---:|
| 인제군 | 24.2 → 8.23km | 30,773명 | 8,604명 (28.0%) |
| 옹진군 | 75.3 → 15.3km | 19,397명 | 7,220명 (37.2%) |

→ 두 사각지대에 응급거점을 1곳씩만 추가해도 약 5만 명(고령 약 1.6만 명)의 응급 접근성이 구조적으로 개선되는 잠재 효과. (거리 단축의 대상 인구이며, 실제 구조 인원·사망 감소 예측이 아니다.)

## 4. 한계

- 주민등록 기준이라 실거주·관광 인구를 반영하지 못한다. 특히 옹진 같은 도서는 실거주가 더 적을 수 있어 수혜 인구는 상한으로 본다.
- 고령비율은 위험지수 가중 항목이 아니라 교차 해석 변수다. 위험지수 순위 자체는 공급측(사고·사망·거리)으로 산출된다.
- 수혜 인구는 "거리 단축의 잠재 대상(시군구 인구)"이지, 구조 인원·사망 감소를 예측하는 값이 아니다.
- 인구 데이터 매칭에서 시 합계·출장소·일부 일반구 등 25개 잉여 행은 시군구 단위 분석 대상이 아니라 제외했다(grid 252개는 전부 매칭).

## 5. 재현

```
py -3.12 backend/scripts/merge_population.py "<고령인구현황 CSV 경로>"   # grid_features_demo.parquet 생성

```
