## 문제 정의

한국 교통사고 골든타임 놓침으로 인한 사망률이 선진국 2배. 사고 후 1시간 내 수술실 도착률 50%. 그러나 "사고 위험 + 응급 도달 시간"을 결합해 잠재 위험지대를 발굴하는 일반인용 서비스는 부재.

기존 사고 다발지도는 단순 빈도만 보여줌. 진짜 위험한 곳은 **사고는 보통인데 응급차가 늦어 사망률 높은 곳** — 통계가 가려둔 잠재 위험지대.

---

## 목표 사용자 / 수혜자

- **일반 시민** (메인) — 우리 동네 응급 사각지대 확인
- **정책결정자/지자체** (보조) — 응급 자원 추가 배치 시뮬레이션
- **심사위원/연구자** (간접) — 발견형 인사이트·정책 implication 평가

단일 웹앱에 두 모드 토글로 통합.

---

## 핵심 기능 / 산출물

**시민 모드**
1. 주소·동/구 검색 → 격자별 위험지수, 응급도착시간, "왜 위험한가" SHAP 설명
2. 전국 위험지도 (격자 색칠, 호버 툴팁)
3. 상위 N개 잠재 위험지대 리스트

**정책결정자 모드**
4. 가상 응급분서·응급실 위치 추가 시뮬레이션 (지도 클릭)
5. 영향 정량 평가 ("X 위치 추가 시 평균 응급도착 Y분 단축, 위험지수 Z% 감소")

**공통**
6. About 페이지 (데이터 출처, 방법론, 한계)

**대회 제출물**
- 시제품 (Streamlit Cloud 배포 링크)
- 기획서 (대회 양식)
- 데모 영상 (1~2분)

## MVP 범위

**포함 (V1)**
- 시민 모드 (검색·지도·SHAP 설명)
- 정책 시뮬레이터 모드
- About 페이지

**V1.1 (시간 남으면)**
- 1km × 1km 격자 세분화
- 추가 변수 (기상, 시간대 세분화 등)

**제외**
- 사용자 계정·로그인
- 실시간 알림
- 네이티브 모바일 앱
- 다국어
- 차량 내비 연동
- 가명정보 결합·안심구역 활용

## 성공 기준

| 단계 | 목표 |
|---|---|
| 1차 | 본선(2차 심사) 진출 = 멘토링 단계 진입 |
| 2차 | 우수상 이상 (300만원+) |
| 스트레치 | 대상 (700만원) |

부수 기준:
- 가점 15/25 확보 (AI 10 + 데이터융합 5)
- 본인 포트폴리오 가치
