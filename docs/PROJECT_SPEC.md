# Project Spec: molit-2026

> 살아있는 스펙. 정해지는 대로 업데이트.
> 미결 항목은 `[정해야 함]`으로 명시 마킹.

---

## 문제 정의

[정해야 함] — 2026 국토교통 데이터 활용 경진대회 제품/서비스 트랙에 출품할 아이디어 미정.

대회가 풀고자 하는 큰 영역:
- "데이터와 AI로 여는 국토·교통 혁신"
- 공공데이터(도로공사·교통연구원·KORAIL·TS·KAC·HUG·KAIA·부동산원 등)를 활용한 정책/제품/서비스

내가 풀 구체적 문제는 브레인스토밍 단계에서 결정.

---

## 목표 사용자 / 수혜자

[정해야 함] — 아이디어 확정 시 함께 정의.

가능한 후보군 (브레인스토밍 시작점):
- 일반 시민 (출퇴근, 여행, 이주 등 생활 교통 사용자)
- 운수업 종사자 (택시·버스·물류)
- 지자체·공공기관 의사결정자
- 부동산·교통계획 전문가

---

## 핵심 기능 / 산출물

대회 요구 산출물:
1. **시제품** (앱, 웹사이트 등 동작하는 데모)
2. **기획서/발표자료** (심사 단계별 제출물)
3. **데이터 활용 근거** (어떤 공공데이터를 어떻게 썼는지)

세부 기능은 아이디어 확정 후 결정. [정해야 함]

## MVP 범위

**포함**:
- [ ] [정해야 함] — 아이디어 확정 후 채움

**제외**:

## 성공 기준

1차 목표: **본선(2차 심사) 진출** — 멘토링 단계 진입
2차 목표: **우수상 이상 입상** (상금 300만원 이상)
스트레치 목표: **대상** (국토교통부장관상, 700만원)

부수 기준:
- 가점 항목 최대한 확보 (AI 활용 10점 + 데이터융합 5점 = 15점 가산 가능)
- 출품작이 본인 포트폴리오로도 의미 있게 남을 것 (교통시스템공학 전공 연계)
